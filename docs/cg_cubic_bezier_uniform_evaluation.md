## Metric Definitions

- **Max Err**: Maximum normalized L2 error across all elements (meters)

### Observations

1. **Lambda sweet spot**: λ=10 consistently gives lowest errors across most mesh sizes

2. **Mesh refinement helps**: Error decreases from ~34m (4×4) to ~9.2m (32×32)

3. **Constraint comparison**:
   - **c1_vertex_only**: Best for small meshes (4×4, 8×8), fastest
   - **c1_edge_only**: Best for large meshes (16×16+), similar accuracy to c1_vertex_edge but ~2.5× faster
   - **c1_vertex_edge**: Slowest (2-3× slower), minimal accuracy gain over c1_edge_only

4. **Scaling** (32×32 mesh at λ=10):
   - c1_vertex_only: 7.9s
   - c1_edge_only: 11.8s
   - c1_vertex_edge: 21.7s

5. **Over-smoothing**: λ≤0.1 causes significant error increase (up to 33m for λ=0.01)
